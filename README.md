# Crickit Kit Gallery

A fan-made cricket kit showcase built with vanilla HTML, CSS, and Vite-powered JavaScript.

## License
MIT

## Features
- Sand-gold light theme and polished dark theme
- Brand filtering for popular kit makers
- Search across the catalog
- Side-by-side compare tray for up to three kits
- LocalStorage persistence for theme and compare picks
- Fan-portal disclaimer and external reference links

## Run locally
```bash
npm install
npm run dev
```
